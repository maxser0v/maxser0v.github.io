Arbitrage Screener - a tool that helps to spot spreads on assets between various crypto exchanges

Arbitrage Screener analyses prices of 200+ cryptocurrencies on well-established exchanges in real-time using their APIs. The screener utilizes 40 threads to ensure that arbitrage opportunities are up-to-date and ready to be executed. The tracked exchanges are: Binance, ByBit, OKX, KuCoin, LBank, CoinEx, BingX, Poloniex, Huobi, Bitget. Unlike other screeners that track only price of an asset, this screener takes into consideration withdrawal and deposit chains to make sure they match, as well as analyses commision for the transfer to ensure that an arbitrage opportunity is profitable. Once the opportunity is spotted, Telegram Bot notifies the user.

Technologies used:

  • Python: aiohttp, sqlite3, asyncio
  • Telegram Bot: Aiogram
  • SQL Database
  • VPS Ubuntu 20.04 set up
