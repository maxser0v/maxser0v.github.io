Обо мне

С раннего детства меня интересовали компьютеры и технологии. Я начал изучать программирование в 12 лет, потому что мне это было безумно интересно. В 14 лет я познакомился с Python и всеми его возможностями для разработки высокоскоростных веб-приложений, а также различных сложных программ. Наряду с этим я изучил HTML, CSS и JavaScript, чтобы иметь возможность создавать привлекательные веб-сайты. Теперь я создаю мощные веб-приложения, сложные системы искусственного интеллекта и просто простые веб-сайты для малого бизнеса.

PYXL - Программное обеспечение, которое помогает Facebook и Google Ads правильно оптимизировать прогнозируемую ценность клиента.

Это не еще одна платформа для аналитики, а нечто совершенно новое: со сложным машинным обучением под капотом, pyxl.ai прогнозирует ценность вашего клиента на основе тысяч параметров и загружает ее на рекламные платформы. Интегрирован с Zoho CRM — самой популярной CRM в мире.

SpeechyText - Приложение для конвертации речи в текст

Устали печатать тонны текста?
Вы получите полноценное приложение Windows для преобразования речи в текст в реальном времени на любом языке. Приложение использует Google Speech-to-Text API, поэтому вам потребуется ключ учетных данных от Google. Приложение позволяет набирать распознанный текст в любом месте Windows. Его можно использовать для ввода текста в Word, веб-браузере или Excel.
Используемые технологии: Python библиотеки, Pyinstaller, Google Speech-to-Text API
